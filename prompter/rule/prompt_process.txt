NOTE: This file defines the process for executing and tracking prompts in the app development.

• execute
  • steps
    • Read current prompt
    • Execute prompt steps
    • Update app files
    • Update app_index.txt
    • Move marker down
  • order
    • Must follow sequence
    • Don't skip prompts
    • Don't reorder steps
  • verify
    • Check all steps done
    • Verify file changes
    • Confirm index update
    • Validate marker move

• track
  • state
    • Current prompt
    • Completed steps
    • Changed files
    • Index updates
  • history
    • Keep prompt order
    • Record file changes
    • Note index updates
    • Track marker moves
  • reset
    • Clear all files
    • Reset app_index.txt
    • Return marker up
    • Clear history

• format
  • prompt
    • Use clear steps
    • Group related tasks
    • Show dependencies
    • List requirements
  • marker
    • Place after prompt
    • Use divider lines
    • Add blank lines
    • Show position
  • index
    • List all files
    • Use space format
    • Sort alphabetically
    • Show structure
