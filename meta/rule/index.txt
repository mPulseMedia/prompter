NOTE: This file defines the rules for formatting index files that list codenames.

1. structure
    a. header
        1. Start with divider line
        2. Include NOTE section
        3. End header with divider line
        4. List codenames with line numbers
        5. End with blank line
    b. numbering
        1. Use two-digit numbers (01-99)
        2. Follow with period and space
        3. Right-align numbers (01., 02., etc.)
        4. No leading zeros for numbers 10+

2. codenames
    a. listing
        1. Sort all codenames alphabetically
        2. Put each codename on its own line
        3. Show full codename for first in group
        4. Show only unique suffix for others
    b. prefix
        1. For matching prefixes:
            a. Replace with spaces
            b. Use one space per character
            c. Keep only unique suffix
        2. For multiple matches:
            a. Use longest matching prefix
            b. Replace all matching characters
            c. Keep only unique suffix

3. alignment
    a. prefix
        1. Compare with line above
        2. For matching prefixes:
            a. Replace with spaces
            b. Use one space per character
            c. Show only suffix
        3. For no match:
            a. Show full codename
            b. Start new group
    b. indentation
        1. First line sets group indentation
        2. Maintain consistent indentation
        3. No extra spaces for alignment
        4. No tabs or multiple spaces
        5. Match prefix character count

4. special
    a. subsets
        1. Put shorter codename first
        2. Show only unique suffix for longer
    b. shared_prefix
        1. Sort by suffix alphabetically
        2. Show only unique suffix
    c. no_match
        1. Write full codename
        2. Start new group
    d. single_word
        1. Show only suffix
        2. Keep group prefix spacing

5. validation
    a. requirements
        1. Number every line
        2. Sort all codenames
        3. No duplicates allowed
    b. format
        1. Show full codename or unique suffix
        2. Match prefix character count
        3. Maintain group indentation
        4. Follow spacing rules exactly
